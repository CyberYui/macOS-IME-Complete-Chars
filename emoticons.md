# 颜文字使用指南

## 一、Mac 原生颜文字功能

macOS 内置了颜文字输入功能，无需任何第三方工具：

**方法一：字符检视器**
1. 在任意文本输入框中，按下 `Control + Command + 空格`
2. 弹出字符检视器，切换到「表情符号」或搜索「颜文字」
3. 点击即可插入

**方法二：触控栏（Touch Bar 机型）**
- 在文本输入时，触控栏会自动显示表情符号建议，点击即可插入

**方法三：自定义短语（本项目推荐）**
- 将下方颜文字添加到 Mac「系统设置 → 键盘 → 文本输入 → 自定义短语」
- 输入对应的短语代码，颜文字会出现在候选列表中，比字符检视器更快

## 三、如何批量导入

本仓库 `plist/符号和颜文字.plist` 已包含部分颜文字和符号，可直接拖入自定义短语使用。

如需制作自己的颜文字plist，推荐借助AI完成，无需手动编写：

> 我想为macOS自定义短语制作一个plist文件，请帮我把以下颜文字生成为标准格式，触发短语统一用「ym」（内容根据自身需求确定）：
> （粘贴你的颜文字列表）
> plist格式参考：
>
> ```xml
> 	<dict>
> 		<key>phrase</key>
> 		<string>颜文字内容</string>
> 		<key>shortcut</key>
> 		<string>触发拼音</string>
> 	</dict>
> ```
> 请输出完整的plist文件内容。

推荐国内可用AI：[豆包](https://www.doubao.com)、[Kimi](https://kimi.moonshot.cn)、[DeepSeek](https://chat.deepseek.com)

**提示**：本项目的 `radicals.plist` 和 `rare-chars.plist` 专注于偏旁部首和生僻字，颜文字内容独立维护在本文件中。
